Librerías del Sistema

Para el correcto funcionamiento de este datalogger, el proyecto requiere instalar las siguientes librerías desde el Gestor de Librerías de Arduino IDE:

  • SPI.h y SD.h: Incluidas por defecto en el IDE. Gestionan el bus SPI de alta velocidad para la escritura en el módulo MicroSD.
  • RTClib (de Adafruit): Utilizada para el manejo de marcas de tiempo. En este prototipo usamos RTC_Millis para emular un reloj en tiempo real mediante el procesador del Arduino, guardando los datos con fecha y hora exacta.

Solución de Hardware: Librería MAX6675_Lento

Al desarrollar este datalogger, encontramos un conflicto físico: la memoria MicroSD monopoliza los pines SPI de alta velocidad (11, 12 y 13). Si conectamos el sensor MAX6675 a estos mismos pines, la señal colapsa y arroja errores de lectura (0.00 °C).

Para solucionar esto y hacer el proyecto replicable, desarrollamos una librería propia llamada MAX6675_Lento. Esta utiliza "SPI por Software", permitiéndonos conectar el sensor a cualquier pin digital libre (en nuestro caso 2, 3 y 10) sin interferir con la tarjeta SD.

¿Cómo usarla en tu proyecto?
1. En tu Arduino IDE, haz clic en el ícono de los tres puntos (o flecha hacia abajo) en la esquina superior derecha y selecciona "Nueva Pestaña".
2. Nombra la primera pestaña MAX6675_Lento.h y pega el código correspondiente.
3. Repite el proceso para crear una segunda pestaña llamada MAX6675_Lento.cpp.
4. ¡Listo! Ya puedes incluirla en tu código principal.
MAX6675_Lento.h
#ifndef MAX6675_LENTO_H
#define MAX6675_LENTO_H
#include "Arduino.h"

class MAX6675_Lento {
  public:
    MAX6675_Lento(int8_t SCLK, int8_t CS, int8_t MISO);
    void begin();
    float readCelsius();
  private:
    int8_t sclk, cs, miso;
    uint8_t spiread();
};

#endif
MAX6675_Lento.cpp
#include "MAX6675_Lento.h"

MAX6675_Lento::MAX6675_Lento(int8_t SCLK, int8_t CS, int8_t MISO) {
  sclk = SCLK;
  cs = CS;
  miso = MISO;
}

void MAX6675_Lento::begin() {
  pinMode(cs, OUTPUT);
  pinMode(sclk, OUTPUT);
  pinMode(miso, INPUT);
  digitalWrite(cs, HIGH);
}

float MAX6675_Lento::readCelsius() {
  uint16_t v;
  digitalWrite(cs, LOW);
  delay(1);
  v = spiread();
  v <<= 8;
  v |= spiread();
  digitalWrite(cs, HIGH);

  if (v & 0x4) {
    return -100.0; // Error: Termopar no detectado
  }
  v >>= 3;
  return v * 0.25;
}

uint8_t MAX6675_Lento::spiread() {
  int i;
  uint8_t d = 0;
  for (i = 7; i >= 0; i--) {
    digitalWrite(sclk, LOW);
    delay(1);
    if (digitalRead(miso)) {
      d |= (1 << i);
    }
    digitalWrite(sclk, HIGH);
    delay(1);
  }
  return d;
}

Código Principal (Datalogger.ino)

Este es el cerebro del proyecto. Está diseñado con arquitectura no bloqueante usando la función millis() en lugar de delay(), lo que garantiza que el Arduino no se congele durante los intervalos de espera. Guarda una lectura exacta en la memoria MicroSD cada 10 minutos.

Datalogger_Principal.ino
#include <SPI.h>
#include <SD.h>
#include <RTClib.h>        
#include "MAX6675_Lento.h" 

// --- Configuración del Sensor de Temperatura ---
int thermoDO  = 3;  
int thermoCS  = 10;
int thermoCLK = 2;
MAX6675_Lento miSensor(thermoCLK, thermoCS, thermoDO);

// --- Configuración de la Tarjeta SD ---
const int chipSelect = 19; 

// --- Configuración del Reloj Virtual ---
RTC_Millis rtc; 

// --- Variables para el control de tiempo (millis) ---
unsigned long tiempoAnterior = 0;
// 10 minutos = 10 * 60 segundos * 1000 milisegundos = 600,000
const unsigned long intervalo = 600000; 

void setup() {
  Serial.begin(9600);
  miSensor.begin(); 
  
  Serial.println("Iniciando sistema...");
  delay(1000);

  if (!SD.begin(chipSelect)) {
    Serial.println("Error SD. Revisa la tarjeta.");
    while (1); 
  }
  
  // Sincroniza la hora con la del PC al momento de compilar
  rtc.begin(DateTime(F(__DATE__), F(__TIME__)));
  Serial.println("Reloj sincronizado. Listo para medir.");
  Serial.println("---------------------------------------------------");

  // Crea el archivo y los encabezados si no existen
  File archivoDatos = SD.open("datos.csv", FILE_WRITE);
  if (archivoDatos) {
    archivoDatos.println("Fecha,Hora,Temperatura(C)");
    archivoDatos.close();
  }

  tomarMedida(); // Medida inicial al encender
}

void loop() {
  // Verificación no bloqueante del intervalo de tiempo
  if (millis() - tiempoAnterior >= intervalo) {
    tiempoAnterior = millis(); 
    tomarMedida();             
  }
}
